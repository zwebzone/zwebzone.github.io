## Lecture 1

Deep learning allows us to handle unstructured inputs without hand-engineering features with less domain knowledge.

For "few-shot learning", leveraging prior experience, learning quickly something new.

For multi-task learning, different tasks need to share some structure. Learn all of the tasks more quickly or more proficiently than learning them independently.
